# Fast and Fine Technical Services FZE

Professional technical services website for Dubai, UAE.

## About

Fast and Fine Technical Services FZE offers professional carpentry, plumbing, HVAC, electrical, and building maintenance services across Dubai.

## Features

- Bilingual support (English/Arabic)
- Responsive design
- Service booking system
- Portfolio showcase
- Contact forms

## Technology Stack

- PHP 8.1+
- MySQL 8.0
- HTML5, CSS3, JavaScript

## Setup

1. Clone the repository
2. Copy `config.example.php` to `config.php`
3. Update configuration with your settings
4. Import database schema
5. Configure web server
